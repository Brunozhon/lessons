Creating a simple HTML page

First, create a doctype.

<!doctype html>

This doctype was for HTML5. Other doctypes include:

Version 4.01:
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01 Transitional//EN" "http://www.w3.org/TR/html4/loose.dtd">
Version 1.0 (XHTML):
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Transitional//EN" "http://www.w3.org/TR/xhtml/DTD/xhtml1-transitional.dtd">

Next, include a html, head, and body element with the doctype like this:

<!doctype html>
<html>
  <head>
    <title></title>
  </head>
  <body>
  </body>
</html>

Congratulations! You have made a very basic HTML document! Next you'll add some content! Now let's review!
